# RTK Authentication Project

This project is built using **React + Vite + Redux Toolkit (RTK)**.

## 🚀 Features
- Authentication system (login/logout)
- State management using Redux Toolkit
- Fast development with Vite

## ⚙️ Tech Stack
- React
- Redux Toolkit
- Vite
- JavaScript

---

## 📦 Default Vite Setup

This project was initialized with Vite and includes:

- @vitejs/plugin-react (uses Oxc)
- Fast Refresh (HMR)
